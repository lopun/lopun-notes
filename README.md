# Lopun Store

Offline MarkDown Note made by GraphQL & React.js

# Demo

- [Deployed By Now](https://lopun-notes-ztekquzacb.now.sh)

## Commands

- start command

```
npm run start
yarn start
```

- development mode

```
npm run dev
yarn dev
```

- build for production

```
npm run build
yarn build
```
